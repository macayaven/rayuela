Interactive Visualizations for the Computational Analysis of Cortázar's Rayuela

These interactive charts accompany our article on using AI to map the hidden structure of Julio Cortázar's Rayuela (1963). Each visualization is built with Plotly and is fully interactive: hover for details, drag to pan, scroll to zoom, and click legend items to toggle layers. 3D charts can be orbited by dragging.
